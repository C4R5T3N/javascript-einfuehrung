In JavaScript haben wir es fast ausschließlich mit Objekten zu tun. Auf jeden »Wert« im Speicher haben wir Zugriff als ein Objekt. Diese Objekte haben meistens gewisse Eigenschaften und Methoden. Diese verhalten sich selbst wieder als Objekte - so entsteht eine Objekthierarchie, die prinzipiell unendlich fortgesetzt werden kann.

Der Browser stellt einem Script eine ungeheure Anzahl von vordefinierten Objekten zur Verfügung. Ein Script nutzt einerseits diese vordefinierten Objekte, indem es Eigenschaften ausliest und Methoden aufruft. Andererseits definiert das Script eigene Objekte. Diese vom JavaScript-Programm selbst eingeführten Objekte werden Variablen genannt.
