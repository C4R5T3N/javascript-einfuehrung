JavaScript: Aufgaben und Anwendungsbereiche

Was ist unter JavaScript zu verstehen?

JavaScript ist eine Programmiersprache, die in Webseiten eingebunden wird. Die JavaScript-Programme, auch Scripte genannt, werden im Web-Browser interpretiert. Das heißt, sie werden in Maschinencode übersetzt und ausgeführt. Da JavaScripte auf dem Rechner des Websurfers ausgeführt werden, spricht man von einer clientseitigen Programmiersprache, um sie von Programmen abzugrenzen, die auf dem Web-Server laufen, wie etwa PHP- oder Perl-Scripte.

Was macht JavaScript und wie funktioniert JavaScript?

JavaScripte haben Zugriff auf das Browserfenster und das darin angezeigte HTML-Dokument. Ihre wichtigste Aufgabe besteht darin, auf Benutzereingaben im Dokument zu reagieren (z.B. klickt der Benutzer auf ein Element oder gibt einen Text in ein Formularfeld ein). JavaScripte können daraufhin Änderungen im gegenwärtig angezeigten HTML-Dokument vornehmen. Selbstverständlich finden diese Änderungen nur im Browser, genauer gesagt im Arbeitsspeicher des Rechners statt, während das Dokument auf dem Web-Server unangetastet bleibt.

Diese Änderungen können sowohl den Inhalt als auch die Darstellung des Dokuments betreffen. Auf diese Weise kann ein Dokument interaktiv und »dynamisch« gestaltet werden. Das bedeutet, es kann auf Benutzereingaben reagieren und sich ändern, ohne dass ein neues Dokument vom Web-Server abgerufen werden muss. Beispielsweise können Zusatzinformationen eingeblendet werden, sobald der Anwender mit der Maus auf ein bestimmtes HTML-Element klickt.

Den lesenden und verändernden Zugriff auf das Dokument regelt das sogenannte Document Object Model (DOM) – dazu später mehr. Der überragender Teil der JavaScript-Programmierung besteht aus dem Umgang mit dem DOM. Wegen dieser zentralen Wichtigkeit wird mittlerweile von »JavaScript/DOM« oder gar »DOM Scripting« gesprochen. Ein älterer, mittlerweile überholter Begriff lautet »Dynamisches HTML«, abgekürzt DHTML. Dies soll Ihnen nur verdeutlichen, dass sich JavaScript in erster Linie darum dreht, mit dem HTML-Dokument zu hantieren, in dessen Kontext ein Script ausgeführt wird.

JavaScript als vollwertige Programmiersprache

Für viele Webautoren ist JavaScript die erste Programmiersprache, mit der sie zu tun bekommen. HTML und CSS sind zwar auch künstliche Rechnersprachen, aber keine Programmiersprachen – es lassen sich damit keine Programme schreiben, die Anweisung für Anweisung abgearbeitet werden.

Was zuerst wie eine spitzfindige Unterscheidung klingt, ist ein Unterschied ums Ganze: Mit HTML lassen sich lediglich Texte auszeichnen und Dokumente aufbauen. Mit CSS werden diese Dokumente formatiert und gelayoutet. In CSS werden sogenannte Deklarationen verfasst, die HTML-Elemente ansprechen und ihnen Formatierungen zuweisen. Eine Programmiersprache funktioniert grundlegend anders und erfordert eine eigentümliche Denkweise und Herangehensweise an Probleme. Die Grundlagen der Programmierung können an dieser Stelle nicht beschrieben werden, sie sind aber essentielle Voraussetzung für den Umgang mit JavaScript.

Bedeutung von JavaScript für das Web

JavaScript ist aus dem heutigen Web nicht mehr wegzudenken und nimmt neben HTML und CSS eine enorm wichtige Rolle ein. Beinahe jede Website verwendet kleinere oder größere JavaScripte.

Zwar existiert die Sprache schon seit über zehn Jahren, wurde stetig fortentwickelt und hat verschiedene Phasen durchlebt. Aber erst in den vergangenen paar Jahren kam es zu einem regelrechten Boom, einer Renaissance und Neubegründung von JavaScript. Während JavaScript vorher eher das »Schmuddelkind« unter den Webtechniken war und seine Daseinsberechtigung oft angezweifelt wurde, explodierte die JavaScript-Nutzung unter dem Schlagwort Ajax. Zudem setzen sich jüngst Theorien durch, die den vernünftigen JavaScript-Einsatz begründen und rechtfertigen.

Heißt das, dass Ihre Website JavaScript verwenden muss, um vollständig zu sein? Nicht unbedingt. Aber viele verbreitete Funktionen lassen sich mit JavaScript vereinfachen und benutzerfreundlicher gestalten und neue Funktionen können hinzukommen.

Besonderheiten von JavaScript

JavaScript nimmt unter den Programmiersprachen einen besonderen Platz ein. Wenn Sie bereits andere Programmiersprachen kennen, werden Sie sicher Gemeinsamkeiten erkennen und Ihr Wissen wiederverwenden können. Doch früher oder später werden Sie auf Eigentümlichkeiten von JavaScript stoßen, die Sie so in keiner anderen im Web verbreiteten Programmiersprache finden werden. Eigentümlichkeiten, deren Beherrschung äußerst nützlich bei der JavaScript-Programmierung ist. Sobald Sie komplexere Scripte schreiben, sollten Sie mit diesen für JavaScript typischen Programmiertechniken Bekannschaft machen – Sie werden sie zu schätzen lernen.

JavaScript hat sehr klein angefangen und wurde nicht auf dem Reißbrett entworfen, um den Anforderungen des breiten Einsatzes zu genügen. Heutzutage beruhen große Webanwendungen auf umfangreichen und hochkomplexen JavaScripten. JavaScript gerät dadurch in das Interesse von Informatikern und Software-Entwicklern, die mit der Sprache experimentieren, deren Grenzen austesten und ihre Erkenntnisse weitergeben. Die Vorteile von JavaScript sowie empfehlenswerte Programmiertechniken sind dadurch erst nach und nach ans Licht gebracht worden - und es ist ein andauernder Prozess, den Sie verfolgen sollten und an dem Sie sich beteiligen können.
